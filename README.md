# ☕ Be My Cafe ☕
## 이윤주 강민지 최보람

### 개발 기간 : 10 월 5 일 ~ 11 월 23 일

### 게임 설명 : 타이쿤 형식의 카페 시뮬레이션 게임으로 플레이어는 정해진 시간 안에 최대한 많은 음료를 제작하여 스코어를 기록해야 함. 카페에서 아르바이트 한 경험이 있는 팀원들로 구성하여 실제 카페와 비슷한 게임을 제작함.
